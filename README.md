# Jayasurya Portfolio Website

A clean and modern personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## 🚀 Project Summary

- **Title:** Jayasurya | Web Developer
- **Tech stack:** HTML, CSS, JavaScript
- **Sections:** Hero, About, Education, Skills, Projects, Contact
- **Goal:** Showcase skills, projects, education, and contact details for internship/job opportunities.

## 🧩 Features

- Responsive navigation bar with hamburger menu (mobile support)
- Animated skill bars (progress values in HTML data attributes)
- Project cards with live/demo and GitHub links placeholders
- Contact section with a form and direct email/LinkedIn/GitHub links
- Smooth layout and modern typography (Google Fonts)

## 📁 Repository contents

- `index.html` - main page markup
- `style.css` - styling for layout, colors, responsive design
- `script.js` - interactive behaviors (nav toggle, smooth scroll, etc.)
- `javascript-39424.png` / `JS.png` - logo/avatar images
- `README.md` - this documentation

## 🛠️ Setup and run

1. Clone repository

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
```

2. Open `index.html` in a browser (double-click or use Live Server)

3. Edit content in `index.html` as needed:
   - Name, headline, bio
   - skill levels and project details
   - contact links

## ✨ Customization

- Update `<title>` in `index.html`.
- Replace the avatar/logo images with your own.
- Add/remove project cards in the `#projects` section.
- Adjust skill percentages in the `#skills` section by setting `data-width` and the text value.
